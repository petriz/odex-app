Política de privacidad

Última actualización: 18 de marzo de 2026

1. Información que recopilamos

ODEX recopila únicamente la información necesaria para el funcionamiento de la aplicación:

  • Datos de pacientes ingresados manualmente por el usuario (nombre, fecha de nacimiento, contacto)
  • Información clínica registrada por el profesional dental (tratamientos, notas, citas)
  • Datos de uso de la aplicación para mejorar el servicio

2. Uso de la información

La información recopilada se utiliza exclusivamente para:

  • Proveer las funcionalidades del sistema de gestión clínica
  • Mejorar la experiencia del usuario
  • Brindar soporte técnico cuando sea solicitado

3. Almacenamiento de datos

Los datos clínicos se almacenan localmente en el dispositivo del usuario mediante una base de datos SQLite segura. No se transmiten datos de pacientes a servidores externos sin consentimiento explícito.

4. Compartición de datos

No vendemos, intercambiamos ni transferimos información personal a terceros. Los datos de pacientes son estrictamente confidenciales y no se comparten con ninguna entidad externa.

5. Seguridad

Implementamos medidas de seguridad técnicas y organizativas para proteger la información almacenada en la aplicación, incluyendo autenticación por contraseña y cifrado de datos sensibles.

6. Derechos del usuario

El usuario tiene derecho a acceder, rectificar y eliminar toda la información almacenada en la aplicación. Para ejercer estos derechos, contacte a: mpl3234@gmail.com

7. Cambios a esta política

Nos reservamos el derecho de actualizar esta política de privacidad. Los cambios se notificarán a través de la aplicación o mediante correo electrónico.

8. Contacto

Para preguntas sobre esta política de privacidad: mpl3234@gmail.com
